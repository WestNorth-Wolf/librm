template<usize MaxBindings = 32, usize MaxConcurrentKeys = 8, bool UseStdFunction = true>
class rm::modules::KeyboardHandler< MaxBindings, MaxConcurrentKeys, UseStdFunction >

键盘输入处理器

模板参数
MaxBindings最大绑定数量（KeyDown和KeyUp各自的上限）
MaxConcurrentKeys最大同时按下的普通键数量
UseStdFunction是否使用std::function作为回调类型（否则使用etl::delegate）

template<usize MaxBindings = 32, usize MaxConcurrentKeys = 8, bool UseStdFunction = true>
void rm::modules::KeyboardHandler< MaxBindings, MaxConcurrentKeys, UseStdFunction >::Update ( u32  keycode)
inline

更新键盘状态并触发相应的回调

参数
keycode当前的keycode，每一位对应一个键的按下状态

处理逻辑：

  1. 计算新按下的键和新释放的键
  2. 处理modifier键状态更新
  3. 对于新按下的普通键，检查是否满足组合键条件
  4. 触发相应的KeyDown/KeyUp回调
